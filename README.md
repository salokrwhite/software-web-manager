# Software Web Manager

统一软件版本管理平台（Go + React + Ant Design）。支持版本发布、灰度/预览、回滚、强更/可选更新、渠道分发与数据分析，并提供 SDK 接入。

## 目录结构
- `backend/` Go API + worker
- `web/` React 管理端
- `sdk/` Go / Node / C# / Java / C++ / Rust / Python SDK
- `deploy/` Docker Compose 与 Nginx
- `docs/` 文档与 OpenAPI
